# Lead Magnet Mastery Agent — Master Index

## System Overview

This agent applies 14 direct response skills across 2 completed phases to build a lead magnet from first principles — from market diagnosis to fully engineered content architecture — before a single word of copy is written.

The system is designed around one principle: **a great lead magnet is an engineering problem before it is a writing problem.** Every skill in Phases 01 and 02 is structural. Phase 03 (copywriting) does not begin until the structure is airtight.

## Output Artifacts — What Phases 01 and 02 Produce

After completing all 14 skills, you have the following working documents:

**From Phase 01:**
- **Strategic Awareness Brief** — awareness level, entry angle, headline frame
- **Emotional Entry Brief** — 4-layer emotional landscape (surface/dominant/suppressed/aspirational)
- **VOM Report** — verbatim market language in 6 categories, resonance-scored
- **Desire Map** — 3-layer desire architecture + false belief chain
- **Competitive Landscape Report** — 8–12 competitor audits + gap opportunity matrix
- **Prospect Portrait** — 10 dimensions + Copy Anchor Statement (read before every copy session)

**From Phase 02:**
- **Promise Statement** — specific 5-component promise, Scorecard ≥ 8/12
- **Specificity-Enhanced Promise** — vague words replaced, Vagueness Watchlist active
- **Value-Friction Audit** — ratio ≥ 2.0, interventions complete, price anchor selected
- **Format Recommendation** — primary format, signal audit, negative signal interventions
- **Bridge Blueprint** — delivery zone, gap statement, desire gradient, bridge statement draft
- **Consumption Architecture Plan** — section sequence, reading rhythm spec, completion prediction
- **Quick Win Specification** — type, full draft, placement, forward-pull bridge
- **Information Scarcity Frame** — 3 versions (opt-in page, opening contract, email promotion)

**What these 14 artifacts enable:**
A copywriter (or Phase 03 of this agent) can now write every word of copy for this lead magnet — opt-in page headline through bridge statement — without guessing. Every copy decision has a specification. Every promise has a scorecard. Every format has a rationale. Every transition has a blueprint.
