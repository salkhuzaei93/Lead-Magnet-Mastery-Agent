# Value-Friction Calculator

## Instructions
Score each driver and friction point honestly.
Use the scoring guide below each item.
Calculate ratio at the bottom.
Any ratio below 2.0 requires intervention before proceeding.

---

## PART 1 — VALUE DRIVERS (score 0–3 each)

**V1 — Promise Specificity**
Does the promise contain specific outcomes, timeframes, conditions, and mechanisms?
- 0 = Vague, generic, could describe any lead magnet in the niche
- 1 = Some specificity but missing 2+ components from the Promise Architecture
- 2 = Specific outcome and mechanism, may lack timeframe or narrow recipient
- 3 = All five promise components present and highly specific
Score: ____

**V2 — Price Anchor**
Is there an explicit anchor communicating what this would cost if not free?
- 0 = No anchor present
- 1 = Vague anchor ("valuable insights")
- 2 = Anchor present but generic ("normally costs $X" without context)
- 3 = Specific, credible anchor tied to a real comparable (consultation rate, course price, time value)
Score: ____

**V3 — Information Scarcity**
Does the content feel impossible or very difficult to find elsewhere?
- 0 = Generic information easily Googled
- 1 = Useful but available from multiple competitors
- 2 = Synthesized or framed in a way that is differentiated
- 3 = Proprietary framework, original research, or experience-derived insight not replicable
Score: ____

**V4 — Format Premium**
Does the format signal quality and effort?
- 0 = Plain text, no design, template-looking
- 1 = Basic formatting, functional but not premium
- 2 = Clean professional design, looks intentional
- 3 = Premium design, 3D mockup, format creates tangibility and high perceived production value
Score: ____

**V5 — Social Proof**
Is there specific, credible evidence that others have found this valuable?
- 0 = No social proof
- 1 = Generic testimonials, no specifics
- 2 = Named testimonials with specific results
- 3 = Multiple named testimonials + opt-in count or result statistics
Score: ____

**V6 — Source Authority**
What authority signals establish the creator as the right person to deliver this?
- 0 = Unknown creator, no credentials, no signals
- 1 = General credential or experience mentioned
- 2 = Specific relevant result or credential that speaks directly to the promise
- 3 = Multiple authority signals (media, specific results, named clients, published work)
Score: ____

**V7 — Presentation Quality**
Does the visual quality of the opt-in page and lead magnet signal premium value?
- 0 = Poor design, stock template, cluttered or unprofessional
- 1 = Functional but forgettable
- 2 = Clean, professional, clearly designed with intention
- 3 = Distinctive, premium design that communicates investment and seriousness
Score: ____

## PART 2 — FRICTION POINTS (score 0–3 each)

**F1 — Email Address Concern**
Is the prospect's concern about email use addressed?
- 0 = Privacy statement present, specific about frequency and content
- 1 = Generic "no spam" statement
- 2 = No privacy statement
- 3 = Context or page design actively raises privacy concern
Score: ____

**F2 — Time Cost Uncertainty**
Does the prospect know how long this will take to consume?
- 0 = Time commitment clearly stated on opt-in page (minutes to complete/consume)
- 1 = Length implied by format but not stated
- 2 = No time signal — prospect must guess
- 3 = Format or copy implies a significant time investment
Score: ____

**F3 — Skepticism Overhead**
Is the prospect's prior experience with disappointing free content addressed?
- 0 = Skepticism directly acknowledged and neutralized with proof
- 1 = Proof present but skepticism not explicitly addressed
- 2 = Neither acknowledgment nor proof — skepticism left unaddressed
- 3 = Copy or design triggers additional skepticism (over-hyped language, big promises, no proof)
Score: ____

**F4 — Complexity Signal**
Does the lead magnet appear accessible and implementable?
- 0 = Simplicity and accessibility explicitly communicated
- 1 = Implied simplicity through format (checklist, short guide)
- 2 = No simplicity signal — complexity is ambiguous
- 3 = Copy or format implies complexity, technical skill, or significant effort
Score: ____

**F5 — Form Friction**
How many fields does the opt-in form require?
- 0 = Email only (one field)
- 1 = First name + email (two fields)
- 2 = Three fields
- 3 = Four or more fields, or includes phone number
Score: ____

**F6 — Page Cognitive Load**
How many competing elements and decisions does the opt-in page contain?
- 0 = Single-purpose page: headline, sub-headline, bullets, form, CTA only
- 1 = Mostly focused with 1–2 minor distractions
- 2 = Multiple competing elements, navigation present, or secondary CTAs
- 3 = Cluttered, multiple decisions, hard to find the primary CTA
Score: ____

**F7 — Technical Friction**
Are there technical barriers to completing the opt-in?
- 0 = Fast load, mobile-optimized, clear CTA, immediate confirmation
- 1 = Minor technical issues unlikely to prevent completion
- 2 = Noticeable issues (slow load, partial mobile problems, unclear next step)
- 3 = Significant barriers (form errors, broken mobile, no confirmation, unclear CTA)
Score: ____

**F8 — Privacy Concern Escalation**
Does the page design or context raise the prospect's privacy guard?
- 0 = Clean, minimal, low commercial signal
- 1 = Standard commercial signals (one tracking pixel, basic cookie notice)
- 2 = Multiple commercial signals, visible ad tech, or retargeting notices
- 3 = Page aggressively signals data collection or commercial intent
Score: ____

## INTERVENTION PRIORITY MATRIX

After scoring, identify interventions by impact vs. effort:

**High Impact / Low Effort (do immediately):**
- Remove unnecessary form fields (F5)
- Add privacy statement (F1)
- State time commitment (F2)
- Add "reason why free" copy (F3)

**High Impact / Medium Effort (do before launch):**
- Add price anchor copy (V2)
- Add specific testimonial (V5)
- Simplify page to remove cognitive load (F6)

**High Impact / Higher Effort (plan for iteration):**
- Upgrade design and 3D mockup (V4, V7)
- Build case studies for proof (V5)
- Establish authority signals (V6)
