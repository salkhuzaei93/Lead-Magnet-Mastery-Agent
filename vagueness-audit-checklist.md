# Rapid Vagueness Audit Checklist

## Instructions
Paste any copy block into this checklist and scan for each category.
Mark every instance found. Then apply the specificity ladder to each.

---

## Category 1 — Unmeasured Outcomes
Scan for these words and flag every instance:

more / better / faster / easier / higher / lower / improved / increased
enhanced / optimized / maximized / transformed / stronger / smarter
larger / smaller / cleaner / richer / healthier / happier / simpler
dramatically / significantly / greatly / substantially / considerably

**For each flagged word, answer:** "How much [word]? By what measure? In what timeframe?"

---

## Category 2 — Floating Timeframes
Scan for:

quickly / soon / rapidly / in no time / instantly / overnight
in days / in weeks / in no time at all / almost immediately
before long / in the near future / very soon / right away / in record time

**For each flagged phrase, answer:** "Specifically, how long? What is the exact or estimated duration?"

---

## Category 3 — Undefined Recipients
Scan for these over-broad identity terms used without qualification:

entrepreneurs / business owners / marketers / coaches / consultants
professionals / people / anyone / everyone / those who / individuals
small businesses / companies / teams / leaders / managers

**For each flagged term, answer:** "Which ones specifically? At what stage? With what prior experience? With what frustration?"

---

## Category 4 — Mechanism Ghosts
Scan for these unanchored method words:

system / method / framework / approach / strategy / technique
formula / blueprint / secret / process / protocol / tool
program / course / training / resource / guide

**For each flagged word, answer:** "Which [word] specifically? How many steps/parts? What does it do, mechanically?"

---

## Category 5 — Proof Vacuums
Scan for unanchored proof language:

proven / tested / works / results / effective / powerful / guaranteed
success stories / case studies / many people / thousands / hundreds
clients / customers / users / members / students / followers

**For each flagged word, answer:** "How many people specifically? What result specifically? In what timeframe? Can a name or case study be attached?"

---

## Universal Watchlist — High-Risk Vague Words
These words appear in almost all first drafts and almost always need upgrading:

```
amazing       incredible    revolutionary  game-changing  life-changing
world-class   cutting-edge  state-of-the-art  next-level  ultimate
powerful      breakthrough  unique         exclusive     proprietary (without definition)
simple        easy          effortless     stress-free   worry-free
various       several       numerous       certain       some
```

## Specificity Ladder Quick Reference

```
RUNG 1:  Category name only
         "more clients"

RUNG 2:  Qualified direction
         "consistently more qualified clients"

RUNG 3:  Approximate quantity
         "2–3 new clients per month"

RUNG 4:  Specific quantity
         "an average of 2.4 new clients in month one"

RUNG 5:  Quantity with condition
         "2–4 new clients in month one for coaches who complete all four steps"

RUNG 6:  Quantity with named evidence
         "2.7 new clients on average in month one — from 23 coaches in Q1"
```

Climb as high as honesty allows. Stop at the highest defensible rung.

---

## Believability Ceiling Quick Test

Read the specificity-enhanced claim from the perspective of your most skeptical prospect.

[ ] A — Completely credible: proceed
[ ] B — Credible but surprising: add qualifying condition or proof reference
[ ] C — Strains belief: pull back one rung, add evidence
[ ] D — Implausible: return to rung 3 and rebuild from honest data

---

## Irregular Number Preference Guide

When choosing between a round estimate and an irregular measurement:

| Round (estimate) | Irregular (measurement) | Use |
|-----------------|------------------------|-----|
| 30% | 34% or 28% | Irregular — reads as measured |
| 50 clients | 47 clients | Irregular — reads as counted |
| $2,000 | $1,847 | Irregular — reads as real |
| 10 lbs | 11 lbs | Irregular — reads as weighed |
| 20 minutes | 23 minutes | Irregular — reads as timed |
| 90% | 91% or 88% | Irregular — reads as calculated |

Exception: If your actual measurement IS a round number, use it. Never make up an irregular number.
